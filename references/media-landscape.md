# GEO 媒体收录地图：国内 vs 海外

> AI 引擎的引用来源直接决定你的 GEO 可见度。把内容发布到正确的平台，比只优化官网重要得多。
> 数据参考：多平台内容分发可使 AI 引用率提升最高 **325%**。

---

## 海外市场（英文 AI 引擎）

目标平台：ChatGPT Search / Perplexity / Google AI Overviews / Bing Copilot / Claude

### Tier 1 - 必投（被 AI 引用频率最高）

| 平台 | 类型 | AI 引用权重 | 操作建议 |
|------|------|-----------|---------|
| **YouTube** | 视频 | ★★★★★ | 研究证实：YouTube 提及是所有外部信号中相关性最强的。发视频解说、教程、产品演示 |
| **Reddit** | 社区讨论 | ★★★★★ | Perplexity 大量引用 Reddit 帖子。参与相关 subreddit 讨论，建立真实用户声音 |
| **Wikipedia** | 百科 | ★★★★★ | AI 知识库核心来源。若达到"可关注度"标准则创建词条；至少在 Wikidata 建实体 |
| **Forbes / TechCrunch / Wired** | Tier-1 科技媒体 | ★★★★★ | 权威媒体报道是 AI 最信任的引用来源，PR 投放首选 |
| **LinkedIn Articles** | 专业内容 | ★★★★ | 品牌账号发深度文章，结合创始人个人账号发帖，增强实体可信度 |

### Tier 2 - 高价值垂直渠道

| 平台 | 类型 | 适合场景 |
|------|------|---------|
| **G2 / Capterra** | 软件评测 | SaaS/B2B 产品必须有完整档案 + 真实用户评价 |
| **Product Hunt** | 产品发布 | 新产品上线获取权威媒体关注和外链 |
| **Crunchbase** | 公司档案 | 填写完整公司信息，AI 经常引用此作为品牌事实来源 |
| **GitHub** | 技术内容 | 开发者工具/开源项目，技术权威信号 |
| **Quora** | 问答 | 信息型内容，回答行业问题建立专家形象 |
| **Medium** | 博客 | 内容转发/重发，扩大内容分发覆盖面 |
| **HackerNews** | 技术社区 | 技术/创业类产品，高质量外链来源 |

### Tier 3 - 行业垂直媒体（按行业选择）

| 行业 | 推荐媒体 |
|------|---------|
| SaaS / Tech | VentureBeat / The Next Web / TechRadar |
| 航空 / 旅行 | Aviation Week / Business Traveller |
| 金融 | Bloomberg / Financial Times |
| 医疗 | MedScape / HealthTech Magazine |
| 零售 / 电商 | Retail Dive / eMarketer |

## 国内市场（中文 AI 引擎）

目标平台：文心一言 / 豆包 / Kimi / 通义千问 / DeepSeek / 讯飞星火

### Tier 1 - 必投（中文 AI 优先引用）

| 平台 | 类型 | AI 引用权重 | 操作建议 |
|------|------|-----------|---------|
| **百度百科** | 百科 | ★★★★★ | 中文 AI 的核心知识库。品牌词创建词条，产品/创始人均可建条目 |
| **知乎** | 问答 | ★★★★★ | 国内 AI 训练数据重要来源。回答行业问题 + 发文章，结构化问答格式优先 |
| **B站** | 视频 | ★★★★★ | 中文等效 YouTube。产品演示、教程、行业解说视频 |
| **36氪** | 科技媒体 | ★★★★★ | 国内 TechCrunch，最受 AI 引用的中文科技媒体之一 |
| **微信公众号** | 图文 | ★★★★ | 大量中文 AI 训练数据来源。有影响力的号发布内容效果倍增 |

### Tier 2 - 高价值垂直渠道

| 平台 | 类型 | 适合场景 |
|------|------|---------|
| **虎嗅** | 深度商业 | 商业模式分析、行业深度报道 |
| **钛媒体** | 科技创业 | 融资新闻、创业故事、产品发布 |
| **少数派** | 软件评测 | 效率工具、App、SaaS 产品评测 |
| **CSDN** | 开发者 | 技术类产品，代码教程，开发者工具 |
| **掘金** | 开发者 | 技术社区，前端/后端/AI 开发者聚集 |
| **品玩 / 爱范儿** | 消费科技 | 硬件、App、消费级 AI 产品 |
| **企查查 / 天眼查** | 公司档案 | 国内 AI 会引用企业工商信息，完善公司档案 |

### Tier 3 - 权威背书渠道（高难度但高价值）

| 平台 | 权重 | 说明 |
|------|------|------|
| **人民网 / 新华网** | ★★★★★ | 国家媒体，AI 信任度极高，但入选门槛极高 |
| **央视新闻** | ★★★★★ | 曝光即成国家背书，中文 AI 高度信任 |
| **中国信通院** | ★★★★ | 技术类产品发布白皮书或被引用 |

## GEO 内容分发优先级（按阶段）

### 第一阶段 - 实体建立（最先做）

海外：
1. Wikidata 词条（可自建，AI 直接读取）
2. Crunchbase 完整档案
3. LinkedIn 公司页完整填写

国内：
1. 百度百科词条
2. 企查查/天眼查信息核准
3. 知乎账号注册并发布第一批回答

### 第二阶段 - 权威内容分发（1个月内）

海外：
- 在 Reddit 相关版块参与讨论（不是发广告，是真实参与）
- 发布 YouTube 产品解说视频（哪怕低成本屏幕录制）
- 向 G2/Product Hunt 提交产品

国内：
- 知乎：发布 5-10 篇专业问答（行业词相关）
- B站：发布 2-3 条产品/教程视频
- 36氪/虎嗅：联系发稿或自媒体入驻

### 第三阶段 - 媒体 PR 放大（季度目标）

海外：
- 投放 TechCrunch / Wired 等 Tier-1 媒体 PR
- 发布行业数据报告，供媒体引用

国内：
- 联系 36氪/钛媒体采访或投稿
- 发布白皮书/行业报告
